# 天气爬虫实践

这是个Scrapy爬虫学习项目.
目的是抓取成都40天的天气
使用的scrapy爬虫框架

相比更为建议的request，beautiful soup
scrapy对复杂任务有更好的适应性，并具备一定的对反爬虫机制的对抗性

这个项目有助于学习一定的网页知识，并熟悉scrapy的使用。
抓取天气的爬虫，对于任何应用而言都是不错的功能，如自用的esp32芯片，51单片机。
联网应用都可以获取天气，进行自定义程序。
后续可能添加数据库，用于一定的后端演练。
数据库暂定已有sqlite。

经测试，可以爬取成都的天气文件,获取到40天的气温数据
但天气网站已经升级了发爬虫措施，天气数据文件已加入robots.txt
各位请减少爬取频率。
